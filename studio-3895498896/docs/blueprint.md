# **App Name**: LeadMaster Landing Page

## Core Features:

- Dynamic Hero Section: Engaging visual display of LeadMaster's value proposition with key metric highlights and a clear call to action.
- Structured Problem & Solution Explanation: Dedicated content blocks to clearly articulate the problem B2B companies face regarding lead generation and how LeadMaster's managed service provides the solution.
- Visual Lead Qualification Breakdown: A clear, comparative section using cards and visuals to differentiate 'what counts' as a delivered lead from 'what doesn't,' building trust and setting expectations.
- Transparent Economic Model Display: Section detailing the 'pay-for-results' prepaid model, emphasizing consumption only upon successful lead delivery.
- Interactive Dashboard Preview: An engaging, visual representation (mockup) of the client dashboard showcasing key metrics like 'saldo disponible', 'leads entregados', and 'conversaciones iniciadas' to illustrate service visibility.
- AI-Assisted Requirement Configuration Tool: An intelligent tool integrated into the activation flow to help users define their precise lead criteria, using AI to provide structured suggestions and validate inputs for their 'encargo comercial'.
- Service Activation and Requirements Form: A clear, guided multi-step form to collect detailed client requirements for service activation, leading directly to engagement.

## Style Guidelines:

- Primary brand color: #75AADB, a professional blue for main elements and emphasis. The background uses #FFFFFF (clean white) with #EDF3F7 (a very light, subtle blue derived from the primary hue) for contrasting sections, enhancing visual rhythm.
- Accent color for Call to Action: #FE6800, a vibrant orange to ensure prominence for primary interaction points.
- Accent color for positive signals: #4B8A44, a stable green used for validated states, success indicators, and positive feedback, aligning with 'lead validado'.
- Text color: #110402, a deep dark brown/black selected for optimal readability and a serious, B2B aesthetic.
- All text (headlines and body) will use 'Montserrat' (sans-serif), chosen for its modern, clear, and professional appeal. Note: currently only Google Fonts are supported.
- Clean, geometric, and minimalist icons will be utilized to complement the modern B2B aesthetic. This includes icons for validated states (checkmarks), rejection states (cross-marks), and simple visual representations for metrics and dashboard elements.
- The layout features a responsive design with a clear sectional structure, including a two-column hero section, and maintains a maximum content width of 1140px. Generous white space ('mucho aire') is used throughout to enhance readability and achieve a premium, uncluttered look. UI elements like buttons and cards will feature soft border-radii, approximately 23px for buttons, contributing to a modern, approachable feel. Strategic contrast between sections will be used to guide the user's eye and organize information.
- Subtle and purposeful micro-interactions and transitions, such as gentle hover effects for interactive elements and soft fades for content loading or section changes, will enhance user engagement without detracting from the landing page's professional focus.
